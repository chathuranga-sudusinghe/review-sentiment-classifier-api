# Review Sentiment Classifier API

A small NLP mini project that trains a sentiment classification model and serves predictions through FastAPI.

## Project Goal
Input review text and return:
- sentiment label
- confidence score

## Planned Stack
- Python
- TensorFlow / Keras
- FastAPI
- pytest
- Docker
- GitHub Actions

## Project Structure
- `app/` - API code
- `scripts/` - training script
- `tests/` - test files
- `artifacts/` - saved model files
- `infra/terraform/` -  IaC placeholder
